How to create website in GitHub

  1. Create local git repo (git init)

  2. Add files (git add "name file or .")

  3. Save changes (git commit -m "commit")

  4. Create repository in GitHub( yourNickName.gitnub.io)

  5. Add files to GitHub (git git remote add origin https://github.com/yourNickName/yourNickName.github.io)
